1. Úvod

Jedná se o nastavení pro fiktivní media přehrávač, který umožňuje různá nastavení.

2. Metodika

Programy: Jako vývojové prostředí jsem použil program Microsoft Visual Studio a prohlížeč Google Chrome. Dále jsem použil tento internetový XSD validátor

Zdroje: Hlavní zdroj, ze kterého jsem pro svou práci čerpal jsou internetové stránky W3Schools. Dále jsem čerpal inspiraci z oblíbeného mediálního přehrávače VLC.
